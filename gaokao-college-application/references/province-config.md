# 省份配置表与年度核验

本文件用于快速判断省份、批次、志愿单位和核验重点。省级规则每年可能调整，以下配置只作为工作起点；正式分析前必须打开对应省教育考试院当年公告复核。

---

## 使用方式

先确定四件事：

1. 省份
2. 年份
3. 批次（普通本科批/提前批/专项/专科等）
4. 招生类别（普通类/艺术类/体育类/春季高考等）

再读取本表匹配 `workflow`：

| workflow | 适用志愿单位 | 主核验重点 |
|----------|--------------|------------|
| `school-major-group` | 院校专业组 | 院校代码、专业组代码、组内专业、调剂范围 |
| `major-school` | 专业（专业类）+学校 | 专业代码、选科要求、计划数、专业位次、体检限制 |
| `traditional-school` | 院校志愿 | 第一志愿权重、专业服从调剂、院校招生章程 |
| `special-batch` | 提前批/专项/综评/艺术体育等 | 报考资格、投档规则、综合分或校测规则 |

## 未知配置流程

当省份、批次或规则不明确时：

1. 查询省教育考试院官网当年“志愿填报工作通知”“招生录取工作意见”“招生计划说明”。
2. 确认志愿单位是“院校专业组”“专业+院校”还是“院校志愿”。
3. 确认普通类、艺术类、体育类、提前批、专项计划是否使用不同规则。
4. 确认志愿数量、投档比例、是否有专业服从调剂。
5. 在回答中标注“规则来源”和“尚未核实项”。

检索关键词：

```text
[省份] 2026 高考 志愿填报 工作通知 教育考试院
[省份] 2026 普通高校招生 录取工作意见 志愿设置
[省份] 2026 普通本科批 院校专业组 志愿
[省份] 2026 专业 学校 平行志愿
```

## 配置输出模板

给用户说明规则时，先输出简短配置摘要：

```text
规则配置（待/已核验）：
- 省份：
- 年份：
- 批次：
- 招生类别：
- 志愿单位：
- 志愿数量：
- workflow：
- 数据来源：
- 尚未核实项：
```

只有 `workflow` 确定后，才进入冲稳保或志愿排序。

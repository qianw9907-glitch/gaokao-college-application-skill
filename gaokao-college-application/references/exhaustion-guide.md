# 院校穷举方法

高考志愿最容易遗漏院校，本文件提供系统性穷举思路。

---

## 穷举原则

**不要靠印象，要靠清单逐一核查**

每个考生都有固定的"心目中的好学校"，但这个印象往往不全面。系统穷举的目标是：**在考生位次可以覆盖的范围内，不遗漏任何一所值得报考的院校和专业组**。

---

## 按院校层次穷举

### 985院校（39所本部+分校区）

**39所985完整名单：**

**985分校区（独立招生或本部代码内设）：**
- 哈工大威海校区（独立代码）
- 哈工大深圳校区（独立代码）

- 山东大学威海校区（独立代码）
- 山东大学青岛校区（本部代码内，看办学地点）

- 北师大珠海校区（独立代码）

**穷举操作：**
1. 列出39所本部+分校区清单
2. 逐一查2025年在目标省份的普通招生组最低位次
3. 筛选位次在考生合理区间内的（冲档差距<5000位，稳档和保底更宽）
4. 对筛出的院校逐一查专业组构成

### 211院校（约115所）

数量较多，建议先按**考生目标方向**筛选，而不是逐一排查：

**筛选逻辑：**
1. 明确考生目标方向（可以是理工类、文史法经类、艺术类等任意方向）
2. 找出该方向有强势专业的211院校类型（如：行业特色院校、综合类院校、师范类院校等）
3. 在该类型院校中，按位次区间筛选可报的院校和专业组
4. 逐一核实专业组构成是否真正匹配考生方向

**批量查询方式：**
让AI列出目标省份某位次区间内的211院校和对应专业组，标注各组主要方向，
考生根据自己的目标方向筛选，再逐一用官方文件核实。

### 双一流（非211）

双一流院校在特定方向往往有强势专业，适合填保底区间。

**穷举方式：**
- 明确考生目标方向
- 找出该方向有双一流学科的非211院校
- 查该院校在目标省份的位次，确认是否在保底区间

### 省属强校

各省都有在本省招生位次较低但实力不错的省属院校，适合保底：
- 省属重点大学
- 行业特色院校（石油/邮电/电力等）

## 按专业方向穷举

当考生有明确专业方向时，按方向穷举比按院校层次穷举更高效。

**通用操作步骤：**

1. **明确考生方向**
   - 理工类：具体到哪个工科/理科方向
   - 文史法经类：具体到哪个文科/社科方向
   - 艺术/体育类：按各省特殊招生流程处理

2. **找出该方向的特色院校**
   - 该方向有国家重点学科/双一流学科的院校
   - 该方向有行业背景的特色院校（如医学/农业/地质/石油/航天等各有对应院校）
   - 向AI询问："[目标方向]方向有哪些有特色的院校，不限层次"

3. **按位次筛选**
   - 在特色院校中，筛选位次在考生合理区间内的
   - 对每所院校逐一查对应省份的专业组位次

4. **核实组内是否真正包含目标专业**
   - 院校名字和专业名字都可能具有误导性
   - 必须用《招生考试之友》确认组内专业清单
   - 警惕"学院名含目标方向但组内专业不匹配"的情况

**特别提醒：**
行业特色院校（如医学类、农业类、地质类、石油类、邮电类、师范类等）
在本行业方向的专业竞争力往往强于同层次综合院校，但就业方向相对集中，
需要考生明确是否愿意进入对应行业。

## 分校区穷举注意事项

分校区往往被忽视，但可能有重要价值：

**查询步骤：**
1. 确认分校区是否在目标省份单独招生（独立代码 vs 本部代码）
2. 查分校区对应专业组的位次
3. 确认分校区办学地点是否接受
4. 核实分校区组内专业构成

**常见误区：**
- 以为分校区位次=本部位次（往往差距巨大）
- 遗漏了本部代码下的分校区专业（需看办学地点标注）

---

## 穷举完成的标志

以下情况视为该层次穷举完成：
- 每所院校逐一查过位次，确认在考生区间内或排除
- 有位次数据的院校，已核实专业组构成
- 分校区已确认招生方式并查过位次
- 新编号组（无历史数据）已标注并估算

---

## 穷举模板（对话指令）

向AI发出穷举请求的标准模板：

```
我需要系统穷举所有可能适合的院校。
考生情况：[省份][科目类别][位次]
目标方向：[专业方向]

请帮我：
1. 列出[院校层次]在[省份]招生的所有院校
2. 逐一查2025年普通招生组最低录取位次
3. 筛选位次在[XX-XX]区间内的院校和组号
4. 标注哪些需要进一步核实专业组构成

排除：中外合作/国家专项/高校专项/联合培养
```

## 历史类院校穷举

历史类和物理类的穷举逻辑相同，但院校池不同。

**历史类重点院校类型：**

| 类型 | 代表院校 | 适合方向 |
|------|---------|---------|
| 综合985/211 | 人大、南开、武大等 | 文史法经管各方向 |
| 财经类院校 | 中央财经、上财、对外经贸等 | 金融/会计/经济 |
| 政法类院校 | 中国政法、西南政法等 | 法学/政治学 |
| 师范类院校 | 北师大、华东师大等 | 教育/汉语言/心理 |
| 外语类院校 | 北外、上外等 | 外语/翻译/国际关系 |
| 新闻传播类 | 中国传媒大学等 | 新闻/广播/传播 |

**历史类特别注意：**
- 部分顶尖985对历史类招生名额极少（如理工科强校只开极少文科组）
- 历史类竞争集中在文史法经管，同等位次下这些方向比物理类更卷
- 部分院校历史类只有综合大类招生，入学后才分专业

**历史类穷举时额外检查：**
```
□ 确认该院校历史类专业组是否存在（部分理工院校不招历史类）
□ 确认组内专业是否符合考生方向
□ 注意历史类位次区间对应的院校层次可能和物理类不同
```

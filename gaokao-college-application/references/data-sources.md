# 数据来源详解与AI查询Prompt模板

所有分析结果仅供参考。最终数据必须以各省教育考试院、官方志愿填报系统、阳光高考平台、各高校招生章程、《招生考试之友》《招生专业目录》等当年官方渠道和官方材料为准。

## 数据来源优先级

```
1️⃣ 官方文件（最高权威，必须核实）
   - 各省《招生考试之友》/《招生专业目录》
   - 各省教育考试院官网投档数据
   - 各校2026年招生章程

2️⃣ 官方平台（可信，建议核实）
   - 阳光高考平台 gaokao.chsi.com.cn
   - 各省教育考试院"数据中心"历年投档线

3️⃣ AI辅助查询（仅供参考，必须核实）
   - 千问高考专区
   - 其他AI工具
```

---

## 数据可靠性判断规则

| 级别 | 来源 | 使用方式 |
|------|------|---------|
| 高可信 | 《招生考试之友》原文 + 官网投档数据 | 直接使用 |
| 中可信 | 千问给出且与官方文件一致 | 使用，标注来源 |
| 低可信 | 千问单独给出，未经核实 | 标注"待核实"，不作为决策依据 |
| 不可用 | 以下情况一律排除 | 重新查询 |

**不可用的数据情形：**
- 年份口径不匹配（2025年组号直接套用为2026年组号）
- 专业录取位次被混用为专业组最低位次
- 学校整体最低位次被混用为某专业组位次
- 标注"AI生成"且未经核实
- 中外合作/国家专项位次混入普通组数据

## AI查询标准Prompt模板

### 位次查询（千问）

```
[院校名][专业组号]20XX年[省份][物理类/历史类]
普通招生最低录取位次是多少？

注意事项：
- 只要普通招生组
- 不要中外合作/国家专项/高校专项/单列医学组
- 给出的是专业组最低位次，不是某个专业的录取位次
- 不要混用学校整体最低位次
```

### 专业组构成查询（千问）

```
[院校名]2026年在[省份][物理类/历史类]
[专业组号]组内包含哪些专业？
选科要求是什么？招生人数各是多少？
是否有中外合作/国家专项的组混入？
```

### 三年趋势查询（千问）

```
[院校名][专业组号]2023年、2024年、2025年
在[省份][物理类/历史类]最低录取位次，
请按院校专业组分别列出。

注意：
- 如果该组号在某年不存在，请注明"该年无此组"
- 不要用其他组的数据替代
- 不要混用学校整体最低位次
- 不要混用中外合作/专项组数据
```

### 批量位次核查（千问）

```
以下院校2025年河南物理类普通招生组最低录取位次，
请逐一确认（只要普通组，不要中外合作/国家专项）：

院校1 | 组号1
院校2 | 组号2
...

如有数据缺失，请注明"暂无可核验数据"，
不要用估算数字填充。
```

### 院校穷举查询（千问）

```
[院校名]2025年在[省份][物理类/历史类]
所有普通招生组的组号和最低录取位次，
逐组列出。

排除：中外合作、国家专项、高校专项、
联合培养、单列医学组。

已知组号（不用重复列出）：[已知组号列表]
```

## 历年数据查询渠道

### 河南省（示例，其他省份类似）

**2025年：**
- 登录 haeea.cn → 数据中心 → 普通高招录取统计
- 选择年份+科类+批次，输入院校代号+专业组号查询

**2024/2023年：**
- 登录 haeea.cn → 找对应年份"本科一批院校平行投档分数线"
- 选择对应科类（2023/2024看"理科"，2025看"物理类"）

**阳光高考平台：**
- gaokao.chsi.com.cn → 历年分数线 → 选省份+年份+院校

**注意：** 各省系统需要考生账号登录，AI工具无法直接访问。
